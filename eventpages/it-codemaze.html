CodeMaze

About the Event

Code Maze is an event which has two levels. The first level consists of a puzzle. When the puzzle is solved, output will be present. Next level is to write the code based on the output. The code is to be written directly in the editor. When the code is compiled, the given output must be displayed. The output must be same and accurate. There is a time limit for the event. Both the levels must be completed within 3 minutes. The outputs will be easy and can be written in basic C-language. The top 2 teams will be awarded with prizes. Event includes fun in solving the puzzle and knowledge in coding.
Event Rules:
* Each team consists of 2 participants.
* The time limit is 3 minutes.
* Both the levels must be completed within the given time-limit.
* The outputs must match(from the puzzle and the editor).
