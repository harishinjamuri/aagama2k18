CPU Collaboration

About the Event

This competition is designed for students to display skills in processor assembly with great precision and speed. All the components must be replaced back on the CPU and to the right positions. Participants will identify the different processor components and arrange them to a working CPU. Follow instructions to put together a Processor parts using the right components and tools.
Event Rules:
• Only one participant.
• Participants will have to assemble the CPU from the available components.
• Breaking of components strictly prohibited.
